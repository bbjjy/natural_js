Button(N.button)

    개요 Overview

    Button(이하 N.button) 은 a, input button, button 과 같은 버튼 요소들을 공통된 모양과 공통된 동작을 가진 버튼을 만들어 주는 컴포넌트입니다.

    API DEMO

    데모에는 모든 옵션 및 기능을 포함하고 있지 않습니다. 전체 기능 목록은 데모 바로 아래 있는 기능 문서들을 참고 바랍니다.

    The demo does not include all options and features. For a complete list of features, please refer to the function documentation directly below the demo.

    생성자 Constructor

    Name Arguments Type Return Description
    N.button - - N.button N.button의 객체 인스턴스를 생성합니다.
    context jquery object - 기본옵션의 context 옵션과 같습니다.
    opts object - 기본 옵션을 object 타입으로 지정 합니다.
    N(obj).button - - N.button

    N.button의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

    N.button과 동작되는 로직은 같으며 내부적으로 N()의 첫번째 인자가 N.button()의 첫 번째 인자로 지정 됩니다.
    opts object - N.button의 opts 옵션과 같습니다.

    기본옵션 Default Options

    Name Type Default Required Description
    context jquery object null ○ 버튼컴포넌트화 할 버튼 요소를 지정합니다.
    size string "medium"

    크기

    smaller, small, medium, large, big
    color string "white"

    색깔

    white, blue, skyblue, gray, green, yellowgreen

    css/natural.ui.css 파일에 다음과 같이 클래스 스타일을 추가 하면 컬러를 간단하게 확장 할 수 있습니다.
    a.btn_컬러명__,input[type='button'].btn_컬러명__,button.btn_컬러명__ {
        color: #컬러값 !important;
        background-color: #컬러값;
        border: 1px solid #컬러값;
        ...
    }
    disable boolean false 초기화시 비활성 여부

    값이 true 이면
    css/natural.ui.css 파일에 정의된
    a.btn_mouseover__, input[type='button'].btn_mouseover__, button.btn_mouseover__
    a.btn_mousedown__, input[type='button'].btn_mousedown__, button.btn_mousedown__
    a.btn_mouseup__, input[type='button'].btn_mouseup__, button.btn_mouseup__
    a.btn_mouseout__, input[type='button'].btn_mouseout__, button.btn_mouseout__
    스타일이 적용 됩니다.

    선언형옵션

    Declarative Options

    Name Type Default Description
    size string "medium"

    크기

    smaller, small, medium, large, big
    color string "white"

    색깔

    white, blue, skyblue, gray
    disable boolean false 초기화시 비활성 여부

    함수 Methods

    Name Arguments Type Return Description
    context - - jquery object 컴포넌트를 적용한 요소를 반환 합니다.
    selector string - selector 문자열을 입력하면 context 옵션으로 지정한 요소에서 선택된 요소를 반환 합니다.
    enable - - N.button 버튼을 활성화 합니다.
    disable - - N.button 버튼을 비활성화 합니다.

    예제 Examples

    1. 선언형 옵션을 적용하여 버튼생성

    
<a id="aTag" data-opts='{ "color" : "white", size : "big" }'>a</a>
<input id="inputButtonTag" type="button" value="input button" data-opts='{ "color" : "gray" }' />
<button id="buttonTag" data-opts='{ "color" : "green", size : "large" }'>button</button>

<script type="text/javascript">
   	N("#aTag, #inputButtonTag, #buttonTag").button();
</script>

    2. 명령형 옵션을 적용하여 버튼생성

    
<a id="aTag">a</a>

<script type="text/javascript">
   	N("#aTag").button({
   		color: "blue",
   		size: "small"
   	});
</script>
