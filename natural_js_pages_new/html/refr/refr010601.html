Notify(N.notify)

    개요 Overview

    Notify(이하 N.notify) 는 사이트 전역 알림 메시지를 지정한 위치에 표시 해 주는 컴포넌트 입니다.

    Alert(N.alert)는 컨텐츠 영역의 메시지를 처리하는 용도이고 N.notify는 사이트 전역의 메시지를 처리 하는 용도 입니다. 때문에 N.alert의 요소(Element)는 주로 document.body.[pageContext] 영역에 생성 되고 N.notify는 document.body 영역에 생성 됩니다.

    API DEMO

    데모에는 모든 옵션 및 기능을 포함하고 있지 않습니다. 전체 기능 목록은 데모 바로 아래 있는 기능 문서들을 참고 바랍니다.

    The demo does not include all options and features. For a complete list of features, please refer to the function documentation directly below the demo.

    생성자 Constructor

    Name Arguments Type Return Description
    N.notify - - N.notify N.notify의 객체 인스턴스를 생성합니다.
    position jquery object - 기본옵션의 position 옵션과 같습니다.
    opts object - 기본 옵션을 object 타입으로 지정 합니다.
    N(obj).notify - - N.notify

    N.notify의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

    N.notify과 동작되는 로직은 같으며 내부적으로 N()의 첫번째 인자가 N.notify()의 첫 번째 인자로 지정 됩니다.
    opts object - N.notify의 opts 옵션과 같습니다.

    다른 컴포넌트의 context 옵션과 비슷 하지만 용도가 다릅니다. context 속성은 내부적으로 사용 되고 메시지 대화상자 요소가 저장 됩니다.
    displayTime number 7

    함수 Methods

    Name Arguments Type Return Description

    msg string - 표시 할 메시지를 입력 합니다.

    메시지 대화상자를 제거 합니다.

    예제 Examples

    1. 메시지를 표시하기 위한 세가지 방법

    
<script type="text/javascript">
	// 옵션을 지정 하면서 메시지 표시
	N.notify(opts).add(msg, url);
	// 옵션 지정 없이 기본 옶션으로 메시지 표시
	N.notify.add(msg, url);
	// 메시지 대화 상자의 위치를 지정 하고 옵션을 지정 하면서 메시지 표시
	N(position).notify(opts).add(msg, url);
</script>

    2. 하단 우측에 메시지 표시, 메시지 클릭 시 페이지 이동

    
<script type="text/javascript">
	N({
		top : 5,
		right : 10
	}).notify().add("Natural-JS API 문서가 업데이트 되었습니다.", "http://bbalganjjm.github.io/natural_js/");
</script>
