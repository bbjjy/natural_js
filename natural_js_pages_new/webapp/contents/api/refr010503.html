DatePicker(N.datepicker)

    개요 Overview

    DatePicker(이하 N.datepicker)는 달력 대화 상자를 입력요소 주변에 표시하여 날짜를 선택하여 입력할 수 있도록 해 주는 컴포넌트입니다.

    API DEMO

    데모에는 모든 옵션 및 기능을 포함하고 있지 않습니다. 전체 기능 목록은 데모 바로 아래 있는 기능 문서들을 참고 바랍니다.

    The demo does not include all options and features. For a complete list of features, please refer to the function documentation directly below the demo.

    생성자 Constructor

    Name Arguments Type Return Description
    N.datepicker - - N.datepicker N.datepicker의 객체 인스턴스를 생성합니다.
    context jquery object - 기본옵션의 context 옵션과 같습니다.
    opts object - 기본 옵션을 object 타입으로 지정 합니다.
    N(obj).datepicker - - N.datepicker

    N.datepicker의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

    N.datepicker과 동작되는 로직은 같으며 내부적으로 N()의 첫번째 인자가 N.datepicker()의 첫 번째 인자로 지정 됩니다.
    opts object - N.datepicker의 opts 옵션과 같습니다.

    기본옵션 Default Options

    Name Type Default Required Description
    context jquery object null ○ 데이트피커를 적용할 입력 요소를 지정합니다.
    monthonly boolean false true 이면 Monthpicker를 표시합니다.
    focusin boolean true 날짜 입력요소에 자동으로 focusin 이벤트를 bind 하여 Datepicker를 표시할 것인지 여부를 지정 합니다.
    shareEle boolean true

    데이트픽커를 선택하는 다이얼로그 요소를 모든 N.datepicker 요소들이 공유할것인지 인스턴스마다 따로 만들것인지 여부를 지정 합니다.

    false 로 지정하면 N.grid 와 연동하여 사용 시 그리드 렌더링과 페이지 반응속도가 느려질 수 있습니다.
    onSelect function null 날짜나 월(monthonly:true)이 선택되었을 때 발생하는 이벤트입니다. 
    onSelect : function(context, selDate, monthonly) {
	// this 는 N.datepicker instance
	// context 는 날짜가 입력되는 입력요소
	// selDate 는 선택한 날짜의 Date 객체
	// monthonly 는 옵션으로 지정한 monthonly 값
}
    onBeforeShow function null 데이트피커 판넬이 표시되기 전에 발생 되는 이벤트 입니다. 
    onBeforeShow : function(context, contents) {
	// this 는 N.datepicker instance
	// context 는 날짜가 입력되는 입력요소
	// contents 는 데이트피커 판넬
}
    onBeforeHide function null 데이트피커 판넬이 닫히기 전에 발생 되는 이벤트 입니다. 
    onBeforeHide : function(context, contents) {
	// this 는 N.datepicker instance
	// context 는 날짜가 입력되는 입력요소
	// contents 는 데이트피커 판넬
}
    onHide function null 데이트피커 판넬이 닫히고 나서(닫히는 효과 완료 후) 발생 되는 이벤트 입니다. 
    onHide : function(context, contents) {
	// this 는 N.datepicker instance
	// context 는 날짜가 입력되는 입력요소
	// contents 는 데이트피커 판넬
}

    jQuery 이벤트

    jQuery Events

    DatePicker 가 표시 될 입력 요소에 jQuery Event 로 다음과 같은 명칭의 이벤트를 바인드 하면 해당 단계에 실행 됩니다.

    컴포넌트 초기화 시 지정한 같은 명칭의 이벤트가 있다면 초기화 시 지정한 이벤트가 먼저 실행됩니다.

    Event Name Description
    onSelect 날짜나 월(monthonly:true)이 선택되었을 때 발생하는 이벤트입니다. 
    $("selector").bind("onSelect", function(e, context, selDate, monthonly) {
	// e 는 이벤트 오브젝트
	// context 는 날짜가 입력되는 입력요소
	// selDate 는 선택한 날짜의 Date 객체
	// monthonly 는 옵션으로 지정한 monthonly 값
});

    selector 는 DatePicker 가 적용 되는 입력 요소를 지정 해 주세요.
    onBeforeShow 데이트피커 판넬이 표시되기 전에 발생 되는 이벤트 입니다. 
    $("selector").bind("onBeforeShow", function(e, context, contents) {
	// e 는 이벤트 오브젝트
	// context 는 날짜가 입력되는 입력요소
	// contents 는 데이트피커 판넬
});

    selector 는 DatePicker 가 적용 되는 입력 요소를 지정 해 주세요.
    onBeforeHide 데이트피커 판넬이 닫히기 전에 발생 되는 이벤트 입니다. 
    $("selector").bind("onBeforeHide", function(e, context, contents) {
	// e 는 이벤트 오브젝트
	// context 는 날짜가 입력되는 입력요소
	// contents 는 데이트피커 판넬
});

    selector 는 DatePicker 가 적용 되는 입력 요소를 지정 해 주세요.
    onHide 데이트피커 판넬이 닫히고 나서(닫히는 효과 완료 후) 발생 되는 이벤트 입니다. 
    $("selector").bind("onHide", function(e, context, contents) {
	// e 는 이벤트 오브젝트
	// context 는 날짜가 입력되는 입력요소
	// contents 는 데이트피커 판넬
});

    selector 는 DatePicker 가 적용 되는 입력 요소를 지정 해 주세요.

    함수 Methods

    Name Arguments Type Return Description
    context - - jquery object 컴포넌트를 적용한 요소를 반환 합니다.
    selector string - selector 문자열을 입력하면 context 옵션으로 지정한 요소에서 선택된 요소를 반환 합니다.
    show - - N.datepicker Datepicker를 표시 합니다.
    hide - - N.datepicker Datepicker를 숨깁니다.

    예제 Examples

    1. DatePicker 생성

    
<input id="datepicker" type="text" />

<script type="text/javascript">
	N("#datepicker").datepicker();
</script>

    2. MonthPicker 생성

    
<input id="datepicker" type="text" />

<script type="text/javascript">
	N("#datepicker").datepicker({
		monthonly : true
	});
</script>

    3. 선언형 생성 : Format 룰로 지정하여 N.form 이나 N.grid 초기화시 자동으로 DatePicker 생성

    
<div id="form">
	<!-- 첫번째 인자로 "date"룰을 지정하고 세번째 인자로 "date" 나 "month"(Monthpicker가 표시 됨) 지정 -->
	<input id="datepicker" type="text" data-format='[["date", 8, "date"]]' />
</div>

<script type="text/javascript">
	N({ datepicker : "20141212" }).form("#form").bind();
</script>
